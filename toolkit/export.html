📤 Export des prix imagiro → Dofus Toolkit

Contournement Cloudflare : exécute le script directement depuis ton navigateur sur imagiro.laboubourse.com

⚠️ Pourquoi cette page ? imagiro.laboubourse.com utilise le Cloudflare Managed Challenge (Turnstile). Ce système bloque Node.js et curl au niveau TLS, même avec un cf_clearance valide. La seule solution fiable est d'exécuter les requêtes depuis un vrai navigateur déjà authentifié.
1

Ouvrir imagiro dans Chrome

Va sur imagiro.laboubourse.com et assure-toi d'être connecté (la page principale doit charger normalement).

2

Ouvrir la console DevTools

Appuie sur F12 → onglet Console

3

Coller et exécuter ce script

Copie le code ci-dessous, colle-le dans la console, puis appuie sur Entrée.


    

  


  

    
4

    

      
Télécharger les fichiers JSON

      

        Le script télécharge automatiquement un fichier imagiro-prices.json
        (et imagiro-craft-data.json si disponible).
        La progression s'affiche dans la console.
      

    

  


  

    
5

    

      
Importer dans le toolkit

      

        Dans le Dofus Toolkit → 💹 Prix HDV → cliquer sur
        📂 Importer un fichier JSON → sélectionner imagiro-prices.json.
        Faire de même pour imagiro-craft-data.json si téléchargé.
      

    

  







  
💎 Coefficients de brisage — Export DofusDude

  
Calcule le coefficient PA de chaque item craftable via l'API DofusDude. À exécuter depuis n'importe quelle page (pas forcément imagiro).


  

    ⚠️ Ce script fait ~4 700 requêtes vers api.dofusdu.de (par batch de 10 avec délai).
    Durée estimée : 2–3 minutes. Laisse le script tourner jusqu'à la fin.
    Le résultat est un fichier imagiro-brisage-coeffs.json à importer dans le toolkit.
  


  

    
1

    

      
Ouvrir une console DevTools (F12) sur n'importe quelle page web

      
Idéalement sur imagiro.laboubourse.com (craft-data déjà accessible).

    

  


  

    
2

    

      
Coller et exécuter ce script

      
Le script récupère craft-data depuis imagiro, puis calcule le coefficient brisage (poids PA) de chaque item via DofusDude.

      
      

    

  


  

    
3

    

      
Importer imagiro-brisage-coeffs.json dans le toolkit

      

        💹 Prix HDV → 📂 Importer un fichier JSON → sélectionner imagiro-brisage-coeffs.json.
        Le toolkit reconnaît automatiquement le format coefficients.
